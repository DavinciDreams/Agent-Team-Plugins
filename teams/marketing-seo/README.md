# marketing-seo

Marketing, SEO, and social media team with content analyzer, writer, SEO specialist, social manager, and content reviewer agents that collaborate through phased workflows.

## Team Members

| Agent | Role | Model | Phase | Color |
|-------|------|-------|-------|-------|
| content-analyzer | Content Analyst | sonnet | discovery | yellow |
| content-writer | Content Writer | sonnet | execution | green |
| seo-specialist | SEO Specialist | sonnet | execution | cyan |
| social-manager | Social Media Manager | sonnet | execution | magenta |
| content-reviewer | Content Reviewer | sonnet | review | red |

## Usage

Activate the marketing-seo team:

```
/marketing-seo:marketing Create a blog post about digital marketing trends for 2024
```

Check team status:

```
/marketing-seo:status
```

## Workflow

### Campaign Development (5 phases)

1. **Discovery** - Content analyzer researches brand voice, audience, competitors, and opportunities
2. **Planning** - Design content approach, ask clarifying questions, get user approval
3. **Execution** - Content writer, SEO specialist, and social manager work in parallel
4. **Review** - Content reviewer checks for quality, brand consistency, and compliance
5. **Summary** - Document all changes and suggest next steps

### Content Creation (5 phases)

1. **Discovery** - Understand content requirements and audience
2. **Planning** - Outline content structure and strategy
3. **Execution** - Create, optimize, and prepare for distribution
4. **Review** - Quality check and brand alignment
5. **Summary** - Final deliverables and recommendations

### SEO Optimization (3 phases)

1. **Discovery** - Analyze current SEO performance and opportunities
2. **Execution** - Implement SEO improvements
3. **Review** - Validate SEO best practices

## Skills

- **content-strategy** - Content types, calendar planning, distribution channels, performance metrics
- **marketing-conventions** - Brand voice guidelines, style guide, tone and messaging, visual identity
- **seo-best-practices** - Keyword research methods, on-page SEO checklist, meta tags optimization, link building strategies
- **social-platforms** - Platform-specific guidelines, character limits and formats, hashtag strategies, best posting times
